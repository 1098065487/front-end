flex属性使用更推荐使用简写形式，即flex: 单值；

flex属性是：flex-grow（放大），flex-shrink（缩小），flex-basis（分配剩余空间前项目大小）三个属性的合并

常用使用场景：（注意一点，flex属性是加在子元素上的）
1.flex: initial  等同于  flex: 0 1 auto，flex属性默认值
当容器内有剩余空间，尺寸不会增长，容器尺寸不足时，尺寸会缩小，子元素尺寸自适应于内容，通常适用于子元素自动按内容缩小宽度，总宽度和父宽度一致

2.flex: none  等同于  flex: 0 0 auto
容器内元素尺寸不会放大和缩小，同时子元素尺寸固定于尺寸，也就是最大内容宽度，通常适用于子元素按内容撑开，父元素宽度不变，表现为最大子元素宽度

3.flex: 0  等同于  flex: 0 1 0%  （不常用，由场景可知）
容器内元素尺寸不会放大，但是会缩小，同时子元素尺寸表现为最小内容宽度，适用于子元素按最小内容宽度展示，即一个文字宽度，总宽度和父宽度一致，表现为最小子元素宽度，实际场景不多

4.flex: 1  等同于  flex: 1 1 0%
容器内子元素可以放大，也可以缩小，尺寸不足时，优先最小化内容尺寸（最小化自身，然后分配剩余空间），也就是在不侵占其他元素位置的情况下，充分利用剩余空间

5.flex: auto  等同于  flex: 1 1 auto  （不常用，由场景可知）
容器内子元素可以放大，也可以缩小，尺寸不足时，优先最大化内容尺寸（最大化自身，然后按比例缩小），多适用于内容不固定时，可以按内容做缩小
