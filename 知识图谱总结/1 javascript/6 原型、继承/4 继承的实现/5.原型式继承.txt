说明一下，es6以前的继承，基本是寄生组合继承的做法，包括class也是这个思想的语法糖，但接下来还会介绍三种继承方法

1.原理：
其实就是自己实现一个Object.create()，创建一个构造函数，将构造函数的原型指向传入的对象，然后new 一个实例并返回，结果是获得一个空对象，原型指向传入的对象
function object(obj) {
  function F() {};
  F.prototype = obj;
  F.prototype.constructor = F;
  return new F();
}

2.优点：
在不用创建构造函数的情况下，实现了原型链继承（父类模板由传入对象决定），减少代码量

3.缺点：
和原型链继承思想相似，缺点也相似
1）实例会共用继承的引用类型实例属性
2）无法直接给父类构造传入参数
