1.javascript是一种弱类型语言（动态语言），不用提前声明变量的类型，类型会在程序运行中被确定，也就是可以用一个变量保存不同类型的值

2.数据类型
最新的ECMAScript标准定义了8种数据类型：
7种原始类型：
Boolean  Null  Undefined  Number  String  BigInt  Symbol
引用数据类型  Object

js变量可以赋两种类型的值，一种是7种原始类型，叫原始值，一种是引用数据类型，叫引用值
原始值存在栈中，变量直接存值，引用值存在堆中，变量存的是指针，是引用地址值

Number类型：
js中number类型都是双精度的浮点值，没有为整数做一个特殊的类型来
number类型还包括：+Infinity  -Infinity  NaN

BigInt类型：
用任意精度去表示整数，可以安全的存储和操作大整数，甚至超过数字的安全整数限制
BigInt是通过整数末尾附加n，或者调用构造函数来创建的
BigInt和number类型不能互换操作，注意根本上是两个类型

String类型：
字符串类型在类C语言中，也归属引用类型，因为其长度可变，在js中，属于原始数据类型
尽管原始数据不可改变，但是对于string类型，可以通过substr()和concat()方法操作数据，注意方法返回的是新字符串

Symbol类型：
符号类型是ES6新定义的类型，符号类型是唯一的且不可修改，可以类比其他语言中的枚举类型
Symbol类型的值可以作为object类型的key值

3.对象，属于引用数据类型
js中，可以将对象看作一组属性的集合，属性可以增减，属性的值可以是任意类型，属性的key可以是string或symbol类型
对象的属性由两种：数据属性和访问器属性
数据属性可以理解成一个键值对，包括键值对本身的属性，是否可写，可遍历，可配置
访问器属性指的是访问器函数（get, set用来存取值的方法）及其特性

对象：键值之间的映射组合
函数：附带可被调用功能的常规对象
数组：使用整数作为键，表示列表或集合的最优选择（与对象相比较）
键控集：Maps, Sets, WeakMaps, WeakSets 把对象的引用当作键
结构化数据：JSON
js内置对象：如Date
