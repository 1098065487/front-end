类中静态方法用static修饰，该方法不会被实例继承（实例不可调用），只能被类直接调用

理解：类相当于实例的原型，所有类中定义的方法，都会被实例继承。静态方法创造了例外

注意：
1）如果静态方法内部使用了this，这里this指向的是类，而不是实例。
2）静态方法可以和非静态方法重名。
3）注意是实例不能继承调用静态方法，子类是可以继承父类的静态方法的，同样是类名调用
4）子类可以继承调用父类静态方法，在子类内部的静态方法中（静态方法this、super指向的都是     类），也可以通过super调用到父类静态方法，super此时指向父类（感觉几乎用不上）

class Foo {
    static bar() { this.baz(); }                       // 对应1
    static baz() { console.log('hello') }         // 对应2
    baz() { console.log('world') }
}

Foo.bar()    // hello
