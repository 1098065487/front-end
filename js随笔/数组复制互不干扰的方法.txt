对数组的复制，一般会是浅拷贝（深拷贝如反序列化，会使数组操作变麻烦），为使浅拷贝的数组互不干扰使用，一般有两种简单的小技巧：

1）slice()，返回一个新的数组，原本是数组的截取，包前不包后，无参数为复制

2）concat()，合并数组，返回一个新数组，原本是多数组的合并，无参数为复制

let a = [1, 2, 3, 4];
let b = a;
b.pop();   // pop()，删除数组最后一项并返回
b // [1, 2, 3]
a // [1, 2, 3]

b = a.slice();
b.pop();
b  // [1, 2]
a  // [1, 2, 3]

b = a.concat();
b.pop();
b  // [1, 2]
a  // [1, 2, 3]

所以，再也不要再用 b = a 的数组复制方式了
