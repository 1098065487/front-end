思想：借助于原型，基于已有对象创建新对象，同时不必因此创建自定义类型(省略了父类型构造)
      其实就是定义一个函数，先创建一个临时性构造函数，指定传入的对象为临时构造的原型，返回      临时构造的实例

这种思想其实就是es5中新增的Object.create()方法的思想，当Object.create()只有一个参数时，就和原型式继承的函数定义相同了。
function createObj(o) {
    function F() {};
    F.prototype = o;
    return new F();
}

缺点：
和原型链继承一样，对父类型引用类型的属性值，所有子类型实例共享。


其实看实现方法可以发现，实际操作和原型链继承是一样的，只不过不用再定义父类型的构造，直接使用父类型的实例。
